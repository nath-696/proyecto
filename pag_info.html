Inversión en IA en salud (2025)

Proyecto: Sistema Inteligente para la Detección Temprana de Cáncer de Mama

Descripción: Actualmente se está desarrollando un sistema innovador basado en Machine Learning para la detección temprana de cáncer de mama mediante el análisis de imágenes de mamografía. El sistema entrena modelos de IA con bases de datos médicas validadas, logrando identificar masas sospechosas con alta precisión.

  • Aplicación web intuitiva para profesionales médicos.
  • Herramientas de IA explicable, visualizando zonas detectadas.
  • Chatbot con IA para resultados, consultas y explicaciones.
  • Sistema como apoyo médico eficiente y en tiempo real.

Objetivo General: Detectar masas sospechosas con alta precisión y sensibilidad, con resultados confiables y visualmente interpretables, mejorando diagnósticos médicos y apoyando intervenciones tempranas.

Objetivos Específicos:

  • Modelos IA (Faster R‑CNN, YOLO, U‑Net) entrenados para precisión y sensibilidad.
  • Infraestructura tecnológica segura con bases de datos e interoperabilidad.
  • Interfaz web y chatbot facilitan interacción e interpretación.
  • IA explicable garantiza transparencia y confianza.

Objetivos Informáticos:

  • Optimización de modelos ML/DL para detección eficiente.
  • Infraestructura segura y en tiempo real para imágenes mamográficas.
